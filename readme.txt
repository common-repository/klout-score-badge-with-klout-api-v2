This plugin allows you to display your Klout score in a elegant way using the Klout API V2.

== Description ==

With a minimalist configuration (basically, just your klout username) you will be able to display your Klout score in a elegant and easy way using the Klout API V2. 

== Installation ==

1. Upload files (`kscore.php`, img folder and 'readme.txt') to the `/wp-content/plugins/` directory
2. Activate the widget through the Wordpress configuration
3. Set up the widget in the sidebar where you want to display your Klout score, enter your klout name. And here you are.


== Frequently Asked Questions ==

= Is this iframe version ?=

No, this plugin is using the Klout API V2 partener so your html syntax will be valid in xHtml.

= Why there is no change when I change settings ? =

Please, refresh your cache system to see changes.

= How to customize this plugin ?=

4. If you need to customize the plugin just go to your plugin directory /klout-score-badge-with-klout-api-v2/kscore.php and feel free to modify it.
